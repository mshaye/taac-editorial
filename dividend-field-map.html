Verdict Box
The assessment box at the top of the report body. This section contains a mix of static framing and data-driven values embedded in generated prose. IMPORTANT: The box has two visual states — green (.verdict-box, default) for a healthy dividend, amber (.verdict-box.caution) for a stressed or at-risk dividend. The CSS class applied to .verdict-box, .verdict-eyebrow, and their colours all flip together based on dividend health.
